# Module Status

This document clarifies which modules are implemented and which are scaffolds.

## Active

- `auth`: register, login, JWT auth middleware, role middleware

## Scaffolded (Not Implemented Yet)

- `users`
- `properties`
- `rooms`
- `reservations`
- `payments`
- `reviews`
- `amenities`
- `admin`

## Team Rule

- Do not treat scaffolded modules as production-ready.
- Before enabling any scaffolded route in `src/routes/index.js`, complete controller/service/model/validation and add endpoint tests.
